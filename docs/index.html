Why Flagged

🔒
100% Local AI
Runs on your machine via LM Studio. No email body ever leaves your hardware. Works with GLM-4, Phi-3, Llama, Mistral — any local model.
🎯
Tunable to Your World
Edit PRIORITIES.md to tell Flagged who you are and what matters. Crypto events, VC intros, podcast requests — define your own signal.
📱
Telegram Alerts
Uses your existing Telegram bot. No new apps. Alerts include sender, subject, score, reason, and preview — everything you need to decide.
📬
Multiple Inboxes
Monitor unlimited Gmail accounts from a single config. One Google Cloud project, one credentials file, separate tokens per account.
👁️
Read-Only
Flagged cannot send, delete, or modify emails. Gmail OAuth is read-only. It watches. You respond.
⚙️
Always On
Runs as a Mac launchd service (Linux/Windows support coming). Boots with your machine, keeps watching while you work.

Quick Start

# Clone and install
git clone github.com/MSanchezWorld/flagged
pip install -r requirements.txt

# Configure
cp config.example.json config.json
# Add Telegram token + Gmail credentials

# Edit your priorities
nano PRIORITIES.md

# Run
python flagged.py
